# Premium Kanban Board

A clean and intuitive Kanban board application for task management.

## Features
- Task organization across multiple columns (To Do, In Progress, Done).
- Drag and drop functionality (if implemented).
- Modern and responsive design.

## Files
- `index.html`: The main board interface.
- `styles.css`: Styling for the board and task cards.
- `script.js`: Core logic for managing tasks and board state.

## Getting Started
Open `index.html` in your web browser to start managing your tasks.
