# Premium Personal Portfolio

A professional and responsive personal portfolio template.

## Features
- Modern hero section with call-to-action.
- Showcase for projects and skills.
- Responsive layout for mobile and desktop.
- Interactive animations and smooth scrolling.

## Files
- `index.html`: The main portfolio page.
- `styles.css`: Custom CSS for branding and layout.
- `script.js`: JavaScript for page animations and navigation.

## Getting Started
To view your portfolio, open `index.html` in your web browser.
