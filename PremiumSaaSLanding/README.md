# Premium SaaS Landing Page

A high-converting landing page for SaaS products.

## Features
- Compelling hero section and value proposition.
- Feature breakdown and benefits.
- Signup and demo request forms.
- Fully responsive design.

## Pages
- `index.html`: The primary landing page.
- `signup.html`: Dedicated signup page.
- `demo.html`: Product demo request page.

## Files
- `styles.css`: Styling for the landing page and forms.
- `script.js`: Interactive elements and form handling logic.

## Getting Started
To view the landing page, open `index.html` in your web browser.
