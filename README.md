# Frontend Projects Collection

This repository contains a collection of high-quality frontend projects. Each project is located in its own directory with its own set of resources.

## Projects

- **[AI dashboard](./AI%20dashboard)**: A modern AI dashboard interface.
- **[PremiumEcommerce](./PremiumEcommerce)**: A full-featured premium e-commerce website.
- **[PremiumKanbanBoard](./PremiumKanbanBoard)**: A sleek and functional Kanban board application.
- **[PremiumPortfolio](./PremiumPortfolio)**: A professional and responsive personal portfolio.
- **[PremiumSaaSLanding](./PremiumSaaSLanding)**: A high-converting SaaS landing page.

## Technologies Used

These projects primarily use:
- HTML5
- CSS3
- JavaScript (ES6+)

## Getting Started

To view any of these projects, simply open the `index.html` file in the respective project folder using your web browser.
