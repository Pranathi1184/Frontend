# Premium E-Commerce Website

A premium and fully-featured e-commerce storefront.

## Features
- Dynamic product listing pages.
- Category-specific views (Men, Women, Arrivals, Collections).
- User profile management interface.
- Responsive design with interactive elements.

## Pages
- `index.html`: The main storefront landing page.
- `arrivals.html`: Showcases the latest arrivals.
- `collections.html`: Displays product collections.
- `men.html`: Category page for men's products.
- `women.html`: Category page for women's products.
- `profile.html`: User profile page.

## Files
- `styles.css`: CSS styling for all pages.
- `script.js`: JavaScript for interactivity and dynamic content.

## Getting Started
To view the e-commerce store, open `index.html` in your web browser.
